# Medical Orchestra Landing Page

Premium landing page for the Medical Orchestra AI Platform.

## Technology Stack
- **Core**: HTML5, Vanilla JavaScript
- **Styling**: Vanilla CSS (Modern CSS with variables, glassmorphism, and responsive design)
- **Build Tool**: Vite

## Features
- **Academic Premium Design**: Dark theme with deep colors, clean typography, and plenty of whitespace.
- **Multi-Agent Architecture Showcase**: Detailed breakdown of the Orchestrator, Pharmacologist, and Skeptic agents.
- **Interactive Elements**: Scroll animations (fade-up), smooth navigation, and glassmorphism effects.
- **SEO Optimized**: Semantic HTML and proper meta tags for search engines and social sharing.
- **Mobile First**: Fully responsive layout optimized for smartphones and tablets.

## Getting Started

### Development
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

### Production Build
To create a production-ready bundle:
```bash
npm run build
```
The output will be in the `dist` directory.

## Deployment Instructions

### Vercel / Netlify
1. Connect your repository.
2. Set the build command to `npm run build`.
3. Set the output directory to `dist`.

### Manual Deployment
Copy the contents of the `dist` directory to your web server (VPS, S3, etc.).
